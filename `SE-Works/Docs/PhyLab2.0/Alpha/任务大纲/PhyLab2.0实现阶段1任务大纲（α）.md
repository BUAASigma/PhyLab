# 任务概述

当前最大的问题是核心功能残缺，急需检测各个已实现的子实验脚本的可用性以及正确性，依照情况下线并修补相应的子实验脚本，并为用户提供可用性信息。前端的用户体验存在一些问题，比如找不到注册页面、实验中心操作繁琐、社区与功能系统不统一等，需要进行改进，并且修改网页主题，带个用户新鲜感。在完成相应的修复后，考虑加入新的实验计算脚本，深化用户的刚性需求体验。
