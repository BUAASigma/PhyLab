## 预习要点
```latex
\lambda =\frac{a\Delta x}{D}
```
## 实验原理
### 1.双棱镜干涉
```latex
\Delta L=\frac{ax}{D}=\left\{\begin{matrix}      这五行是一个公式
k\lambda  \\ 
\frac{2k+1}{2}\lambda \\
\end{matrix}\right.
(k=0,\pm 1,\pm2,...)  

\Delta x=\frac{D}{a}\lambda 
\lambda =\frac{a\Delta x}{D}
a=\sqrt{b{b}'}
D=S+{S}'
\lambda =\frac{\Delta x\sqrt{b{b}'}}{S+{S}'}
```
### 2.劳埃镜干涉
```latex
\Delta x=\frac{D}{d}\lambda 
\lambda =\frac{d}{D}\Delta x
```
### 实验内容
```latex
\lambda =\frac{\Delta x\sqrt{b{b}'}}{S+{S}'}
\Delta (u)=\Delta ({u}')
\frac{\Delta b}{b}=\frac{\Delta {b}'}{{b}'}
```
